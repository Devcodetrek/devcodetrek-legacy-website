## About the Project

This repository contains an earlier version of the official **DevcodeTrek** website.

It was developed as a modern, single-page portfolio and corporate presentation website using React, Vite and Tailwind CSS. The project has been preserved as part of DevcodeTrek's development history and frontend portfolio.

> This is a legacy project and is no longer used as the primary DevcodeTrek website.

## Features

- Responsive single-page interface
- Animated loading screen
- Dynamic navigation with active section tracking
- Animated hero section and typing effect
- About and professional profile sections
- Technology and skill presentation
- Portfolio and certificate sections
- Statistics cards with animations
- Social media links
- Contact interface
- Locally stored visitor comments
- SEO metadata management
- Mobile-friendly navigation
- Smooth transitions with Framer Motion

The project is preserved for archival and portfolio purposes.

Current limitations:

- The contact form is a frontend demonstration and does not send emails.
- Visitor comments are stored only in the browser's local storage.
- No backend API or database is included.
- Some portfolio actions and external links are placeholders.
- Authentication and administration features are not included.

## Installation

Clone the repository:

```bash
git clone https://github.com/Devcodetrek/devcodetrek-legacy-website.git
cd devcodetrek-legacy-website
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the address shown in the terminal, usually:

```text
http://localhost:5173
```

## Production Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

The generated production files will be located in:

```text
dist/
```
